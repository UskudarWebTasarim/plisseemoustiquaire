# Plissee Moustiquaire

Boutique en ligne de moustiquaires plissées premium pour fenêtres et portes-fenêtres. Protection anti-insectes élégante et efficace pour votre habitat.

## Caractéristiques

- Moustiquaires plissées de haute qualité
- Protection contre tous les insectes volants
- Design moderne et discret
- Installation professionnelle
- Livraison dans toute l'Europe

## Technologies Utilisées

- Next.js 16
- React
- Tailwind CSS
- TypeScript

## Développement

Pour démarrer le serveur de développement:

```bash
npm run dev
```

Ouvrez [http://localhost:3003](http://localhost:3003) dans votre navigateur.

## Déploiement

Ce projet est optimisé pour le déploiement sur Vercel.
